# [Taxonomy Index](https://development.azurecurve.co.uk/classicpress-plugins/taxonomy-index/)
![Plugin Banner](/assets/pluginimages/banner-1544x500.png)

# Description

Displays index of Categories/Tags or other taxonomy types using taxonomy-index Shortcode.

Example use: `[taxonomy-index taxonomy='category' slug='ice-cream']`

Alternative `[ti]` shortcode can also be used

Taxonomy can be set to category or tag or other taxonomy post type.

This plugin is multisite compatible; each site will need settings to be configured in the admin dashboard.

# Installation Instructions

* Download the latest release of the plugin from [GitHub](https://github.com/azurecurve/azrcrv-taxonomy-index/releases/latest/).
* Upload the entire zip file using the Plugins upload function in your ClassicPress admin panel.
* Activate the plugin.
* Configure relevant settings via the configuration page in the admin control panel (azurecurve menu).
